# Portafolio Personal - TP1

Este repositorio contiene el trabajo práctico obligatorio 1: una Landing Page de Portafolio Personal (HTML y CSS).

Breve descripción:
Este proyecto es una página de presentación personal con secciones sobre mí, proyectos (tarjetas), habilidades, formulario de contacto y una lista de películas favoritas. Está pensada para ser responsiva y accesible.
